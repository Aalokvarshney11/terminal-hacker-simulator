# Terminal Hacker Simulator

A browser-based terminal hacking game built entirely with JavaScript.

## Hackathon Constraints

![Hackathon Constraints](assets/Constraints.jpeg)
## Overview

Terminal Hacker Simulator is a command-line style puzzle game where players act as a cyber agent attempting to breach a secure system.

The objective is to discover hidden passwords, bypass security layers, and complete all three levels before time runs out.

The game features a retro hacker-terminal experience with animated output, command-based interaction, limited attempts, and progressive difficulty.

---

## Features

* Terminal-style gameplay
* Typewriter animation
* 3 security levels
* Randomized passwords
* Countdown timer
* Limited attempts system
* Dynamic password hints
* Password reveal on failure
* Automatic game restart
* Command-based interaction
* Pure JavaScript implementation

---

## Commands

| Command         | Description                         |
| --------------- | ----------------------------------- |
| scan            | Scan the target system              |
| hint            | Reveal password clues               |
| status          | Display current game status         |
| access password | Attempt to unlock the current level |

Example:

```text
access shadow
```

---

## Gameplay Flow

```text
SCAN
  ↓
HINT
  ↓
ACCESS PASSWORD
  ↓
NEXT LEVEL
```

Complete all three levels to successfully finish the mission.

## Hackathon Compliance

This project was developed according to the hackathon constraints:

* JavaScript Only
* No Imports
* Terminal-Based Gameplay
* Browser Execution Supported
* All UI Generated Dynamically Through JavaScript
* No External CSS File
* No UI Markup Inside HTML Body

The HTML file is used only as a minimal loader and mount point. All interface creation, layout, and styling are generated programmatically using JavaScript.

# Terminal Hacker Simulator

A browser-based hacking simulation game built using pure JavaScript, HTML, and CSS.

## Hackathon Constraints

![Constraints](assets/Constraints.jpeg)

## Overview

Terminal Hacker Simulator is a command-line style puzzle game where players act as a cyber agent attempting to breach a secure system.

Players must:

* Scan the system
* Gather password clues
* Crack passwords across 3 security levels
* Beat the countdown timer
* Avoid running out of attempts

The game uses a retro hacker-terminal interface with animated text output for an immersive experience.

---

## Features

* Terminal-style UI
* Typewriter text animation
* 3 progressive security levels
* Randomized passwords
* Limited attempts system
* Countdown timer
* Dynamic hint system
* Automatic game reset after win/loss
* No external libraries
* Pure JavaScript implementation

---

## Commands

| Command         | Description                  |
| --------------- | ---------------------------- |
| scan            | Scan the target system       |
| hint            | Reveal password clues        |
| status          | Show current game status     |
| access password | Attempt to unlock the system |

Example:
